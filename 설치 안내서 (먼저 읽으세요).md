# 환경의학 기출앱 — 폰에 설치하기

이 폴더는 **폰(안드로이드·아이폰)에 아이콘으로 설치**해서 오프라인으로도 쓸 수 있는 앱입니다.
설치는 딱 두 단계예요: **① 이 폴더를 웹에 한 번 올리기(GitHub Pages)** → **② 폰에서 주소 열고 "홈 화면에 추가"**.
한 번 올려두면, 그 주소를 아는 사람은 누구나 각자 폰에서 설치할 수 있습니다.

> ⚠️ 폰에 "아이콘 앱"으로 설치하려면 웹 주소(https)가 반드시 필요합니다.
> 압축만 풀어서 폰에서 여는 방식으로는 홈 화면 아이콘 설치가 안 됩니다(브라우저·기기 제약).
> PC에서 테스트만 할 거라면 맨 아래 "부록"을 보세요.

---

## 이 폴더에 든 것

```
index.html                  ← 앱 본체 (문제 520개 + 해설이 모두 안에 들어있음)
manifest.webmanifest        ← 앱 이름·아이콘 정보 (건드리지 마세요)
sw.js                       ← 오프라인 캐싱 담당 (건드리지 마세요)
robots.txt                  ← 검색엔진 노출 차단
icons/                      ← 앱 아이콘들
설치 안내서 (먼저 읽으세요).md   ← 지금 이 문서
```

> 이 파일들은 **폴더 구조 그대로** 올려야 합니다. 이름이나 위치를 바꾸면 아이콘·오프라인이 안 될 수 있어요.

---

## 1단계 — GitHub Pages에 올리기 (5분, 무료)

### 준비: GitHub 계정
- github.com 에 가입(무료). 이미 있으면 로그인.

### (1) 저장소 만들기
1. 오른쪽 위 **`+`** → **New repository** 클릭.
2. **Repository name**: 아무 이름이나 (예: `hwankyung`). 소문자·영어로.
3. **Public** 선택 (Pages 무료 사용을 위해). — 공개지만 주소를 아는 사람만 접근하고, 검색엔진 노출은 이미 차단해뒀습니다.
4. **Add a README file** 체크.
5. **Create repository** 클릭.

### (2) 파일 올리기 (드래그&드롭)
1. 만들어진 저장소 화면에서 **Add file** → **Upload files** 클릭.
2. **이 폴더 안의 파일과 `icons` 폴더를 통째로** 드래그해서 올립니다.
   - `index.html`, `manifest.webmanifest`, `sw.js`, `robots.txt`, 그리고 `icons` 폴더(그 안 이미지들 포함)까지.
   - (`설치 안내서 .md`는 올리든 말든 상관없어요.)
3. 아래 **Commit changes** 초록 버튼 클릭.

### (3) Pages 켜기
1. 저장소 위쪽 **Settings** 탭 클릭.
2. 왼쪽 메뉴에서 **Pages** 클릭.
3. **Build and deployment → Source**를 **Deploy from a branch**로 둡니다.
4. **Branch**에서 **main** 선택, 폴더는 **/ (root)** 선택 → **Save**.
5. 잠시(1~2분) 기다리면 위쪽에 주소가 뜹니다:
   **`https://<내아이디>.github.io/hwankyung/`**
   이게 앱 주소예요. (뜰 때까지 새로고침 몇 번 해보세요.)

> ✅ 이제 이 주소를 카톡 등으로 공유하면, 받은 사람도 아래 2단계로 각자 설치할 수 있습니다.

---

## 2단계 — 폰에 설치하기 (각자 1번)

### 아이폰 (iPhone / iPad)
1. **Safari로** 위 주소를 엽니다. (크롬 말고 꼭 Safari)
2. 아래 **공유 버튼**(□에 화살표) 탭.
3. **홈 화면에 추가** 탭 → **추가**.
4. 홈 화면에 "환경의학" 아이콘 생김. 이제 앱처럼 열고, 오프라인도 됩니다.

### 안드로이드 (갤럭시 등)
1. **Chrome으로** 위 주소를 엽니다.
2. 오른쪽 위 **⋮ 메뉴** 탭.
3. **앱 설치** 또는 **홈 화면에 추가** 탭 → 확인.
4. 홈 화면에 아이콘 생김. 오프라인도 됩니다.

> 첫 실행 때 한 번은 인터넷이 필요합니다(내용을 폰에 저장). 그 뒤로는 인터넷 없이도 열려요.

---

## 앱을 나중에 고쳤을 때 (업데이트)

문제·해설을 수정한 새 `index.html`을 받으면:
1. GitHub 저장소에서 **Add file → Upload files**로 새 `index.html`을 올려 덮어씁니다(Commit).
2. `sw.js` 파일을 열어 맨 위 `hwankyung-v1`을 `hwankyung-v2`처럼 숫자만 올려서 다시 올립니다.
   → 이렇게 하면 설치한 사람들 폰에서 다음 접속 때 새 버전으로 갱신됩니다.

---

## 부록 — PC에서 그냥 테스트만 하고 싶을 때

폰 아이콘 설치가 아니라 **PC에서 열어보기만** 할 거라면:
- `index.html`을 더블클릭해도 문제 풀이는 됩니다(데이터가 다 안에 있어서).
- 단, 이렇게 열면 "설치"·오프라인 캐싱 기능은 작동하지 않습니다(브라우저 보안 정책상 웹 주소에서만 됨).
- 그 정도면 충분하면 그냥 더블클릭해서 쓰시면 됩니다.

---

## 자주 묻는 것

- **주소가 공개인데 괜찮나요?** 검색엔진 노출은 차단해뒀습니다(robots.txt + noindex). 주소를 아는 사람만 들어옵니다. 그래도 완전 비공개를 원하면 아는 사람에게만 주소를 공유하세요.
- **아이폰에서 크롬으로 하면 안 되나요?** 아이폰은 홈 화면 추가가 Safari에서만 제대로 됩니다. 꼭 Safari로 여세요.
- **돈이 드나요?** GitHub Pages는 무료입니다.
